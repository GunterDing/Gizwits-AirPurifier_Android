智能云空调
=============

Gizwits Air Condition Android Demo App

这是一款使用XPGWifiSDK的开源代码示例APP，可以帮助开发者快速入手，使用XPGWifiSDK开发连接机智云的物联APP。

项目依赖和安装

▪	XPGWifiSDK的jar包和支持库
登录机智云官方网站http://gizwits.com的开发者中心，下载并解压最新版本的SDK。
下载后，将解压后的目录拷贝到复制到 Android 项目 libs 目录即可。
